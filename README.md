
## ChordMini Web

Browser-based chord recognition. Works fully offline with all computation localized.

### Credits

- [ChordMiniApp](https://github.com/ptnghia-j/ChordMiniApp) by [ptnghia-j](https://github.com/ptnghia-j): Provided the initial inspiration for the project.
- [librosa](https://librosa.org): Ported core algorithm used for performing Hybrid Constant-Q Transforms to extract audio features.
- [ISMIR2019-Large-Vocabulary-Chord-Recognition](https://github.com/music-x-lab/ISMIR2019-Large-Vocabulary-Chord-Recognition): Ported logic for advanced chord transcription based on deep learning models and chord decomposition.
